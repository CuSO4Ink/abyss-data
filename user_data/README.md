# User Data

This is the Obsidian-first active knowledge surface for Abyss.

Use this directory as an Obsidian vault, or open it as a folder inside an existing vault.

Recommended structure:

```text
Home.md
_inbox/
_templates/
_attachments/
directions/
projects/
decisions/
summaries/
promoted/
```

Conventions:

- Use Markdown `.md` files as primary notes.
- Use YAML frontmatter when helpful.
- Use Obsidian wiki links like `[[Project Name]]`.
- Store note assets under `_attachments/` and link them relatively.
- Keep `Home.md` as the default entry point.
- Put promoted archive excerpts in `promoted/` with provenance.
